# Agency Workflow Examples

Manage multiple client brand profiles and generate validation reports.

## Files

| File | Description |
|------|-------------|
| `multi_client.py` | Validate content for multiple clients in one run |
| `report_generator.py` | Generate a summary report from validation results |
| `client_configs/client_a.json` | Brand profile for Client A |
| `client_configs/client_b.json` | Brand profile for Client B |

## Usage

```bash
python multi_client.py
python report_generator.py
```
