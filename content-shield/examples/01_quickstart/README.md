# Quickstart Examples

Get up and running with Content Shield in under five minutes.

## Files

| File | Description |
|------|-------------|
| `validate_single.py` | Validate a single piece of content against all shields |
| `validate_batch.py` | Validate multiple content items in one call |

## Prerequisites

```bash
pip install content-shield
```

## Usage

```bash
python validate_single.py
python validate_batch.py
```

Both scripts use local-only shields (no API keys required).
