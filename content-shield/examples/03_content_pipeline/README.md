# Content Pipeline Examples

Integrate Content Shield into a content generation-and-validation pipeline.

## Files

| File | Description |
|------|-------------|
| `generate_and_validate.py` | Generate content then validate it before publishing |
| `batch_processor.py` | Read content from JSON file, validate, and report |
| `sample_content.json` | Sample content items for batch processing |

## Usage

```bash
python generate_and_validate.py
python batch_processor.py
```
