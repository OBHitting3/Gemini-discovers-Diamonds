Iron Forge CLI Suite

A tool you run from your computer's command line to start, build, and manage software projects — like a digital workshop foreman.

So... What Exactly Do I Have?

You have a command-line tool — a program that you control by typing text commands into a black window on your computer (called a "terminal" or "command prompt"), rather than clicking buttons on a visual screen.

Think of it like this

Imagine you walk into a workshop. Instead of picking up each tool by hand, you have a foreman standing there. You just tell him what you need:

"Start a new project." — He sets up a workbench with all the folders and settings you need.
"Build it." — He assembles everything into a finished product.
"Check my work." — He inspects everything for problems.
"Show me the settings." — He shows you exactly how things are configured.

Iron Forge is that foreman. You type a short command, and it does the heavy lifting.

It is not a website, a phone app, or something with buttons to click. It lives in the terminal — that dark window developers type into. But it's polished: it shows colored text, tables, progress indicators, and a nice banner when you use it.

What Can It Do?

Iron Forge has 5 main commands, each one like a different department in your workshop:

🏗️

Init (Start a Project)

Creates a brand-new project folder with everything organized for you — like getting a fresh desk with labeled drawers. You can choose from 3 starter layouts: Default, Minimal, or Library.

🔨

Build (Assemble It)

Takes your source files and packages them into a ready-to-use output folder. Like taking raw lumber and turning it into a finished shelf. Can clean up old builds first.

🔍

Check (Inspect It)

Runs a series of quality inspections: Is the project set up correctly? Are the settings valid? Any code style issues? Gives you a pass/fail report card.

⚙️

Config (Settings)

View, change, or look up any setting for your project. Like opening the control panel — you can see every knob and switch and adjust them.

📋

Info (Status Report)

Shows you a full report: what version of Iron Forge you're running, what your computer has installed, and details about your current project.

How Do I Actually Use It?

Since this is a command-line tool (not a website), here's how you'd get it running on your computer:

  1. Open your terminal.
    On Mac: search for "Terminal" in Spotlight.
    On Windows: search for "Command Prompt" or "PowerShell" or "Git Bash".
    On Linux: you probably already know this one.
  2. Make sure Python is installed.
    Type python3 --version and press Enter. You need Python 3.10 or newer. If it says "not found," you'll need to install Python from python.org.
  3. Download or clone the project.
    If you have the code on GitHub, type:
    git clone https://github.com/OBHitting3/Gemini-discovers-Diamonds.git
    Then enter the folder: cd Gemini-discovers-Diamonds
  4. Install the tool.
    Type: pip install -e .
    This makes the ironforge and forge commands available on your computer.
  5. Start using it!
    ironforge --version — See the banner and version number.
    ironforge init my-project — Create a new project.
    ironforge --help — See all available commands.
